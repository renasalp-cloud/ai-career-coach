## About

AI Career Coach is an AI-powered application designed to help users improve their CVs and prepare for their target careers.

Users can upload their CV in PDF format, and the system performs a comprehensive AI analysis to evaluate the document. Based on the analysis, the application identifies strengths, weaknesses, and areas for improvement, while providing personalized recommendations to make the CV more competitive for the user's desired role.

The long-term goal of this project is to become an intelligent career assistant that helps users improve not only their CVs but also their technical skills, career roadmap, and job readiness.
